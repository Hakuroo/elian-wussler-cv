# Elian Mijael Wussler — Interactive Resume
A coded resume and small portfolio hub built to show how I approach digital production, web implementation, QA thinking, and AI-assisted workflow organization.

## Live Resume

[Open live CV](https://hakuroo.github.io/elian-wussler-cv/)

## Positioning

This resume is built around a junior digital production profile:

* Digital production support
* Design operations
* Basic web implementation
* Asset adaptation
* Mockup and resize workflows
* QA checklists
* Ticket-style documentation
* AI-assisted workflows with human review

The goal is not to present a full-stack or advanced AI engineering profile, but to show how web, logic, and AI tools can support clearer production workflows.

## Features

* Recruiter View as the default clean CV layout
* Creative View with coded interactions and particle canvas
* English / Spanish language toggle
* Print / Save PDF button
* Responsive layout for desktop and mobile
* Tooltip-based skill context
* GitHub and live demo links
* Print-focused CSS for a cleaner PDF version
* Reduced-motion support for accessibility

## QA / Production Mindset Demonstrated

This project was built to demonstrate how I approach digital production work:

* Keep information structured and easy to review
* Turn unclear inputs into clearer follow-up tasks
* Use checklists to reduce production errors
* Keep final outputs human-reviewed
* Separate recruiter-friendly content from creative interaction
* Maintain a clean print/PDF version
* Use AI tools as support, not as a replacement for review

## How to Run Locally

Open `index.html` directly in a browser.

For a local server, you can also use:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
